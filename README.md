# 🚀 React + NestJS + Docker Challenge

## 📌 Instalación
1. Clonar el repositorio
2. Ejecutar `docker-compose up --build`
3. Acceder a:
   - Backend: `http://localhost:5000/api`
   - Frontend: `http://localhost:3000`

## 📌 Funcionalidades
- **Backend (NestJS)**
  - Manejo de contenidos
  - Subida de imágenes
  - Guardado en base de datos
  - Servidor con Docker

- **Frontend (React)**
  - Listado de contenidos
  - Subida de imágenes
  - Integración con API

- **Infraestructura**
  - Nginx como proxy
  - Docker Compose para gestión
  - Volumen persistente para imágenes

## 📌 Uso
- Agregar contenido
- Subir imágenes
- Ver imágenes en la app

## 📌 Comandos útiles
- **Limpiar Docker**: `docker system prune -a`
- **Reiniciar servicios**: `docker-compose down && docker-compose up --build`
